# INVEST-X: Cyber Investigation Intelligence System

INVEST-X is an AI-powered tool designed to assist law enforcement agencies in analyzing cybercrime data and extracting meaningful investigation insights.

## 🚀 Features
- Entity extraction (phone numbers, emails, URLs)
- Risk scoring (Low / Medium / High)
- Structured investigation output
- Case storage for analysis
- Ongoing development for case linking and visualization

## 🧠 Purpose
To reduce manual effort in cyber investigations and help investigators quickly identify important information and patterns from unstructured data like FIRs, chat logs, and emails.

## 🛠️ Tech Stack
- Backend: FastAPI (Python)
- Frontend: React / HTML
- Database: MongoDB
- NLP: spaCy
